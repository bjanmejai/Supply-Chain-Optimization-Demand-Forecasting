# 🚀 Supply Chain Optimization & Demand Forecasting

Demand Forecasting & Inventory Optimization for FMCG Supply Chain
Project Overview
This project builds an end-to-end Demand Forecasting and Inventory Optimization System specifically designed for FMCG and Food products (e.g., Dabur, Farmley, Anmol, Credent). It helps companies predict future demand accurately and optimize inventory levels to reduce stockouts and overstock, especially for seasonal and perishable goods.
Objectives

Develop a multi-product demand forecasting model
Calculate optimal inventory parameters (Reorder Point, Safety Stock, EOQ)
Simulate scenarios for seasonal/perishable items
Provide actionable recommendations via an interactive dashboard
Demonstrate measurable business impact (lower holding costs & stockouts)

Why This Dataset?
The High-Dimensional Supply Chain Inventory Dataset is excellent for this project because:

It contains daily granular data for multiple SKUs (products)
Includes real-world supply chain features: sales, current inventory, lead times, promotions, reorder points, stockouts, and costs
Supports both forecasting and inventory optimization parts perfectly
Allows simulation of cold-chain/perishable scenarios common in FMCG/Food industry
High-dimensional nature makes it realistic for job discussions with analytics/supply chain teams

Dataset Download Link:
https://www.kaggle.com/datasets/ziya07/high-dimensional-supply-chain-inventory-dataset
Tech Stack

Language: Python
Forecasting: Prophet (primary), with options for ARIMA/XGBoost
Optimization: Heuristics + PuLP
Visualization: Plotly, Matplotlib, Seaborn
Dashboard: Streamlit
Environment: Google Colab

Deliverables

Jupyter Notebook with full analysis and modeling
Interactive Streamlit Dashboard (forecasts + inventory recommendations)
Performance metrics (MAE, RMSE, cost reduction %)
Documentation and business insights
